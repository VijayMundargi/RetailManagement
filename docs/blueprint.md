# **App Name**: Storefront MVP

## Core Features:

- Dashboard: Interactive dashboard with clear charts for sales, inventory, and customer trends.
- Basic Product Catalog: Simple interface to add, edit, or delete products manually.
- Basic POS: Point of Sale interface with manual discount application and invoice generation.
- Customer Management: Customer profiles can be created and viewed, with purchase history
- Branch Configuration: Store branch configuration.
- AI Store Optimizer: AI-powered insights tool which can analyze sales and product data, incorporating branch-specific details where appropriate, and suggest adjustments to the store configuration.

## Style Guidelines:

- Primary color: Muted blue (#6495ED) for a calm and reliable feel.
- Background color: Light gray (#F0F8FF), providing a clean backdrop.
- Accent color: Soft purple (#9370DB) to highlight interactive elements.
- Body and headline font: 'PT Sans', sans-serif, for a modern, approachable feel suitable for both headings and body text.
- Use consistent and clear icons for navigation and key functions.
- Clean, grid-based layout with sufficient white space to ensure readability.
- Subtle transitions for loading states and form submissions.
